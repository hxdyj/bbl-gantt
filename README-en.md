## Introduction

[`bbl-gantt`](https://www.npmjs.com/package/bbl-gantt) is a JavaScript Gantt chart library. not dependent on any specific framework.

## Features

- Framework-Agnostic, can be integrated with any Web framework
- Out of the box, easy to get started, rich configuration, clear documentation
- Styling and functionality can be customized through configuration items
