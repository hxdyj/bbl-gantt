## 简介

[`bbl-gantt`](https://www.npmjs.com/package/bbl-gantt) 是一个基于 SVG 的甘特图库，不依赖任何特定框架。

## 特性

- 框架无关, 可与任何 Web 框架集成
- 开箱即用，上手简单，配置丰富，文档清晰
- 样式以及功能都可通过配置项进行自定义
