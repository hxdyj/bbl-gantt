# Introduction

[`bbl-gantt`](https://www.npmjs.com/package/bbl-gantt) is a simple, easy-to-use, customizable JS Gantt chart library that does not depend on any specific framework.

## Features

- Lightweight, easy-to-use, customizable
- Support for duration mode
- Customizable styles
