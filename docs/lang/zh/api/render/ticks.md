# TicksRender

TicksRender 类，负责渲染 ticks 部分

## 构造函数

### constructor

参数：`(gantt: Gantt, renderer: Render)`

构造函数，传入 Gantt 实例和渲染器实例

## 实例属性

### gantt

Gantt 实例

### renderer

Render 实例

### g

Ticks 部分的 G 实例

### gText

Ticks 文字的 G 实例

## 实例方法

### render

渲染整个 Ticks

### clear

清除 Ticks 部分

### destroy

销毁 Ticks 部分
