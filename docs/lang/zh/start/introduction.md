# 简介

[`bbl-gantt`](https://www.npmjs.com/package/bbl-gantt) 是一个简单、易用、可定制的 JS 甘特图库，不依赖任何特定框架。

## 特性

- 轻量、易用、可定制
- 支持 duration 模式
- 支持自定义样式
